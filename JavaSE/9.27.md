- 旋转矩阵关键式子

  ```java
    public void rotate(int[][] matrix) {
          int n = matrix.length ;
          int [][] matrix_new = new int[n][n];
          for (int i=0; i< n; ++i){
              for (int j=0; j<n; ++j){
                  matrix_new[j][n-i-1] = matrix[i][j];
              }
          }
          for (int i=0; i< n; ++i){
              for (int j=0; j<n; ++j){
                  matrix[i][j] = matrix_new[i][j];
              }
          }
      }
  ```

- 先水平翻转再主对角线翻转

```java
public void rotate(int[][] matrix){
    int n = matrix.length;
    for(int i=0; i<n/2; ++i){
        for(int j=0; j<n; ++j){
            int temp = matrix[i][j];
            matrix[i][j] = matrix[n-i-1][j];
            matrix[n-i-1][j] = temp;
        }
    }
    for (int i=0; i< n; ++i){
            for (int j=0; j<n; ++j){
                int temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }
    }
}
```
